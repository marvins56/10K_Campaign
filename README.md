# 10K CAMPAIGN

The project involves developing modules for handling campaign-related functionalities in a .NET application, including commands, queries, and entity configurations.

Delivering a scalable and maintainable solution for managing fundraising campaigns, leveraging modern .NET technologies and best practices. By focusing on DDD principles, CQRS architecture, and robust testing, the application ensures reliability, performance, and flexibility in handling diverse campaign management scenarios.
